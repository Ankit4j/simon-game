# simon-game
simon game
A game of remembering patterns.
There are four colours. 
when starting the game a colour will be highlighted. 
Then press that colour.
After that again a colour will be highlighted then press the first colour and then press the current colour.
Each time a colour is highlighted then remember colours patterns from first to current and click on colours in that order.
This is live at https://ankit4j.github.io/simon-game/
